## controls:

 add cell    : left mouse-click  
 remove cell : lleft moust-click
 zoom in     :  numpad '+'  || scroll up  
 zoom out    : numpad '-'   || scroll down    
 speed up    : numpad '8'  
 slow down   : numpad '9'  
 pause       : spacebar
