# Conways GameOfLife

NOTE: Project in progress.  
A live server is needed to run this project.

## controls:

 add cell    : left mouse-click  
 remove cell : left moust-click  
 zoom in     :  numpad '+' or scroll up  
 zoom out    : numpad '-'  or scroll down    
 speed up    : numpad '8'  
 slow down   : numpad '9'  
 pause       : spacebar
